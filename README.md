# 照片自動矯正器 (Image Leveler)

這是一個前端工具，讓使用者上傳任意一張照片後，系統會分析邊緣的主要傾斜角度，並自動旋轉影像以讓主體看起來水平，同時提供微調滑桿與下載連結。

## 核心流程
- **上傳照片**：透過 HTML 檔案欄位取得圖片，並在畫布內顯示原始效果。
- **梯度分析**：將圖片縮圖到 512px 內，在 Canvas 上計算 Sobel 邊緣，利用梯度方向統計出最常見的角度，視為主體傾斜。
- **旋轉並下載**：使用估算角度旋轉 canvas，並生成 `download` 連結讓使用者保存修正後的結果，亦可使用滑桿微調±15度。

## 使用方式

1. 在瀏覽器中開啟 `index.html`。
2. 點選「📷 選擇照片」上傳一張任意照片。
3. 系統會自動計算傾斜角並顯示已矯正的畫面，角度會在頁面下方顯示。
4. 若需要微調，可拖動滑桿（±15°）修正細部偏差。
5. 按「下載結果」即可取得旋轉後的圖檔。

## 驗證機制

- **角度資訊**：畫面下方會顯示自動偵測角度與滑桿調整量，可確定圖片是否已經水平。
- **預覽比較**：左側 canvas 顯示原始畫面，右側 canvas 顯示旋轉後的結果，方便肉眼比對。

透過這個簡單 Demo，可以快速驗證自動旋轉的準確度與下載流程，也能進一步整合到教育或親子互動場景中。
